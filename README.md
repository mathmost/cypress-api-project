# cypress-api-project
1. 安装好npm后，本地新建一个目录，cd到该目录<br/>
  npm init    						-> 生成package.json配置文件<br/>
  npm install cypress --save			-> 安装cypress包
2. 启动<br/>
    方式一
      &nbsp&nbsp使用 node_modules\.bin\cypress open
    方式二<br/>
      &nbsp&nbspnpx cypress open
    <br/>
    方式三<br/>
      &nbsp&nbspyarn run cypress open
